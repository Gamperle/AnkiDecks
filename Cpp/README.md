# C++ Anki Decks

Flashcard decks covering C++ programming from fundamentals to modern C++ features.

## Available Decks

| File | Description |
|------|-------------|
| `cpp_basics.txt` | Syntax, types, pointers, references, control flow |
| `cpp_oop.txt` | Classes, inheritance, virtual functions, RAII |
| `cpp_stl.txt` | Standard Template Library: containers, algorithms, iterators |
| `cpp_modern.txt` | C++11/14/17/20 features: auto, lambdas, smart pointers, move semantics |
| `cpp_memory.txt` | Memory management, heap vs. stack, new/delete, smart pointers |

## Format

```
Front (question or term)[TAB]Back (answer or definition)
```
