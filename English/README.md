## Topic Index

### Beginner

#### 1. Hardware & Computer Basics
- Components: CPU, RAM, storage, motherboard, GPU, PSU
- Peripherals: monitor, keyboard, mouse, printer, scanner
- Actions: boot, shut down, restart, plug in, upgrade
- Units: byte, kilobyte, megabyte, gigabyte, terabyte

#### 2. Software & Operating Systems
- Types: application, operating system, firmware, driver, utility
- Actions: install, uninstall, update, patch, configure, launch
- Concepts: license, open source, proprietary, freeware, shareware
- OS terms: desktop, taskbar, file manager, terminal, shell

#### 3. Files & Data
- File operations: create, save, delete, copy, move, rename, compress, extract
- File types: text file, binary, executable, archive, image, spreadsheet
- Storage: directory, folder, path, file extension, file size
- Data: backup, restore, sync, corrupt, recover, format

#### 4. Internet & Web Basics
- Networking: browser, URL, domain, IP address, DNS, bandwidth
- Web: website, web page, hyperlink, download, upload, streaming
- Communication: email, attachment, CC/BCC, spam, phishing
- Security basics: password, authentication, encryption, firewall, VPN

#### 5. Basic Programming Concepts
- Core: variable, constant, data type, string, integer, boolean, array
- Control flow: condition, loop, iteration, branch, switch, break
- Functions: function, method, parameter, argument, return value
- Structure: class, object, instance, property, attribute, constructor
- Errors: bug, error, exception, syntax error, runtime error, debug

#### 6. Development Tools & Workflow
- Tools: IDE, editor, compiler, interpreter, debugger, linter
- Version control: repository, commit, push, pull, merge, branch, clone
- Build: compile, link, build, deploy, release, dependency
- Documentation: comment, documentation, README, changelog, specification

### Intermediate

#### 7. Software Engineering Terms
- Design: architecture, design pattern, refactoring, abstraction, encapsulation
- Process: agile, scrum, sprint, backlog, user story, acceptance criteria
- Quality: code review, technical debt, maintainability, scalability, clean code
- Collaboration: stakeholder, requirement, scope, milestone, deadline, deliverable

#### 8. Databases & Data Management
- Concepts: database, table, row, column, record, field, schema, index
- Operations: query, insert, update, delete, join, aggregate, filter, sort
- Types: relational, NoSQL, key-value store, document store, graph database
- Administration: migration, seed, backup, replication, sharding, partitioning

#### 9. Networking & Infrastructure
- Protocols: TCP, UDP, HTTP, HTTPS, FTP, SSH, SMTP, DNS, DHCP
- Components: router, switch, gateway, load balancer, proxy, reverse proxy
- Concepts: latency, throughput, bandwidth, packet, port, socket, handshake
- Cloud: virtual machine, container, instance, cluster, region, availability zone

#### 10. Testing & Quality Assurance
- Types: unit test, integration test, end-to-end test, regression test, smoke test
- Concepts: test case, test suite, assertion, mock, stub, fixture, coverage
- Process: test-driven development (TDD), acceptance testing, boundary testing
- Results: pass, fail, flaky test, false positive, false negative, edge case

#### 11. DevOps & CI/CD
- Pipeline: continuous integration, continuous delivery, continuous deployment
- Concepts: build pipeline, artifact, staging, production, rollback, canary release
- Tools: container, orchestration, infrastructure as code, monitoring, alerting
- Practices: blue-green deployment, feature flag, A/B testing, health check

#### 12. APIs & Integration
- Concepts: API, endpoint, request, response, status code, payload, header
- Styles: REST, GraphQL, RPC, webhook, WebSocket, polling
- Data formats: JSON, XML, YAML, CSV, serialization, deserialization
- Auth: API key, OAuth, token, bearer token, rate limiting, throttling

### Pro

#### 13. Software Architecture
- Patterns: microservices, monolith, event-driven, serverless, hexagonal
- Concepts: coupling, cohesion, separation of concerns, single responsibility
- Distributed: eventual consistency, idempotency, circuit breaker, saga pattern
- Communication: synchronous, asynchronous, message queue, pub/sub, event bus

#### 14. Security Vocabulary
- Threats: vulnerability, exploit, zero-day, attack vector, threat model
- Attacks: injection, cross-site scripting (XSS), denial of service, brute force
- Defense: hardening, penetration testing, audit, compliance, incident response
- Cryptography: hash, salt, cipher, certificate, public key, private key, signing

#### 15. Cloud & Infrastructure
- Services: IaaS, PaaS, SaaS, FaaS, managed service, serverless
- Scaling: auto-scaling, horizontal scaling, vertical scaling, elasticity
- Reliability: high availability, fault tolerance, disaster recovery, failover, SLA
- Cost: pay-as-you-go, reserved instance, spot instance, cost optimization

#### 16. Performance & Optimization
- Metrics: response time, throughput, latency, percentile (p50, p95, p99)
- Problems: bottleneck, memory leak, deadlock, race condition, starvation
- Solutions: caching, indexing, lazy loading, eager loading, connection pooling
- Analysis: profiling, benchmarking, load testing, stress testing, flame graph

#### 17. Technical Communication & Documentation
- Writing: specification, proposal, RFC, ADR, postmortem, runbook
- Meetings: standup, retrospective, demo, sync, kick-off, alignment
- Descriptions: trade-off, constraint, assumption, caveat, prerequisite, blocker
- Feedback: approve, reject, request changes, suggest, propose, escalate

#### 18. Common Verbs & Phrases in Tech
- Actions: deploy, provision, bootstrap, spin up, tear down, deprecate, sunset
- Debugging: reproduce, isolate, trace, diagnose, root cause, workaround, hotfix
- Collaboration: onboard, hand off, pair programming, rubber duck debugging
- Status: out of scope, work in progress, done, blocked, on hold, deprecated

### Expert

#### 19. Compiler & Language Theory
- Compilation: lexical analysis, tokenization, parser, AST, code generation, optimization
- Type systems: strong/weak typing, static/dynamic typing, type inference, generics, covariance
- Functional: monad, functor, closure, currying, higher-order functions, tail recursion
- Memory: garbage collection algorithms, reference counting, mark-and-sweep, generational GC

#### 20. Operating Systems & Low-Level Concepts
- Kernel: kernel space, user space, system calls, context switch, scheduling
- Processes: fork, exec, IPC, shared memory, semaphore, mutex, signals, pipes
- Memory: virtual memory, page tables, TLB, page faults, MMU, swapping
- Devices: device drivers, interrupts, DMA, block devices, character devices, file descriptors

#### 21. Distributed Systems & Consensus
- Consensus: Paxos, Raft, leader election, quorum, Byzantine fault tolerance
- Consistency: CAP theorem, linearizability, causal consistency, vector clocks, CRDTs
- Messaging: exactly-once/at-least-once/at-most-once delivery, dead letter queue, outbox pattern
- Streaming: stream processing, event windows, watermarks, checkpoints, consumer groups

#### 22. Kernel & Hardware-Level Vocabulary
- Kernel internals: slab allocator, buddy system, OOM killer, cgroups, namespaces
- Concurrency: spinlock, RCU, memory barriers, atomic operations, CAS, false sharing
- Tracing: eBPF, XDP, ftrace, kprobe, uprobe, tracepoints, perf
- Security: speculative execution, side-channel attacks, Meltdown, Spectre

#### 23. Formal Methods & Correctness
- Verification: formal verification, model checking, temporal logic (LTL, CTL)
- Contracts: invariants, preconditions, postconditions, Hoare logic, design by contract
- Testing: property-based testing, fuzzing, symbolic execution, abstract interpretation
- Analysis: static analysis, data flow analysis, control flow graph, call graph

#### 24. Advanced Networking & Protocols
- OSI model: all 7 layers and their functions
- TCP internals: three-way handshake, congestion control, window size, SACK, Nagle
- Modern protocols: QUIC, HTTP/2, HTTP/3, gRPC streaming, SSE, WebSocket
- Infrastructure: BGP, OSPF, SDN, NFV, VXLAN, overlay/underlay networks

#### 25. Data Engineering & ML Ops Vocabulary
- Pipelines: data ingestion, transformation, cleansing, lineage, governance
- Storage: data lake, lakehouse, data mesh, columnar formats, schema registry
- ML Ops: model serving, model drift, experiment tracking, feature store, ML pipeline
- Processing: batch, real-time, stream processing, DAG, workflow orchestration

#### 26. Advanced Adjectives, Adverbs & Phrases
- System qualities: resilient, brittle, granular, coarse-grained, idempotent, atomic
- Performance: low-latency, high-throughput, CPU-bound, I/O-bound, memory-intensive
- Lifecycle: GA, pre-release, release candidate, LTS, end of life, deprecated
- Descriptions: comprehensive, negligible, crucial, transient, persistent, interoperable
