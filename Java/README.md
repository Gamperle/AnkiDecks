# Java Anki Decks

Flashcard decks covering Java programming concepts from beginner to advanced.

## Available Decks

| File | Description |
|------|-------------|
| `java_basics.txt` | Syntax, data types, operators, control flow |
| `java_oop.txt` | Classes, interfaces, inheritance, polymorphism |
| `java_collections.txt` | List, Set, Map, Queue, iterators |
| `java_concurrency.txt` | Threads, synchronization, Executors, CompletableFuture |
| `java_jvm.txt` | JVM internals, garbage collection, classloading |

## Format

```
Front (question or term)[TAB]Back (answer or definition)
```
